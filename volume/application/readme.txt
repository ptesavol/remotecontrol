==== Picture Viewer ====

This application displays pictures from the Helsingin Sanomat online news service. The Big screen sandboxed application is required to be installed on the edge. If it is not already installed it will be installed by the edge automatically.

Below are instructions how clients can show pictures on the Big screen.

  * Clients connect their mobile device to the Spaceify wireless network.
  * Client starts his/hers web browser and browses to address <nowiki>hs.fi</nowiki>.
  * The pictures on the web page have green bars on top and bottom of the pictures. Clicking these pictures shows them on the screen.
